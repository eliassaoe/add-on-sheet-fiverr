LinkedIn Profile Finder

This add-on helps you find LinkedIn profile URLs for your contacts using LinkFinder AI.

Getting Started

Step 1: Get your API key from linkfinderai.com
Step 2: Configure your API key in Settings (Add-ons > LinkedIn Profile Finder > Settings)
Step 3: Prepare your spreadsheet with columns for:
  • Full Name (required)
  • Company Name (optional but improves accuracy)
  • LinkedIn URL (output column)
Step 4: Open the sidebar and configure columns, then click "Find LinkedIn Profiles"

How It Works

The add-on processes each row in your selected range and:

  • Combines the name and company fields
  • Sends a request to LinkFinder AI
  • Writes the LinkedIn profile URL to your output column
  • Shows "Not found" if no profile is discovered
Note: Processing includes a small delay between requests to respect API rate limits. For large datasets, this may take several minutes.

Tips for Best Results

  • Use full names (first and last name)
  • Include company names whenever possible
  • Ensure names are spelled correctly
  • Start with row 2 if you have column headers
